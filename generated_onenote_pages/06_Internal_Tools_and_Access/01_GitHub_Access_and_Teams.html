GitHub Access and Teams

How repository access maps to GitHub teams.

Purpose

GitHub Access and Teams is the tool access guide maintained by Developer Platform for engineers and team leads. It covers how repository access maps to GitHub teams. Access is granted through GitHub teams, never to individuals directly.

Key Facts

  • Access is granted through GitHub teams, never to individuals directly.
  • Each repository has a CODEOWNERS file that defines required reviewers.
  • Write access to infra repositories requires security approval.
  • Outside collaborators are reviewed every quarter.

Request Flow

  1. Request to join the relevant GitHub team in the Access Portal.
  2. Wait for the team owner to approve.
  3. Verify you can clone and open a draft PR.
