Software Installation Requests

Installing software outside the standard image.

Purpose

Software Installation Requests is maintained by Endpoint Engineering for employees and service desk agents. This IT support procedure installing software outside the standard image.

Key Facts

  • Pre-approved software installs instantly from the Self-Service Portal.
  • Non-standard software needs a license owner and a security review.
  • Engineers can install developer tools from the approved Homebrew/winget list without a ticket.
  • Unapproved software is blocked by endpoint policy.

Support Process

  1. Check the Self-Service Portal first.
  2. For other software, open a request with a business reason.
  3. Provide license details for paid software.
