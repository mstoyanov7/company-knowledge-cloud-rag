Internal CLI Setup

Installing the 'nw' internal command-line tool.

Overview

Internal CLI Setup is maintained by Developer Platform for software engineers joining a project. This project setup record installing the 'nw' internal command-line tool.

Ownership

Owner teamDeveloper Platform
Audiencesoftware engineers joining a project
SystemsInternal Registry, Okta SSO, Secrets Vault

Key Facts

  • The nw CLI bootstraps projects, reads vault secrets, and runs environment checks.
  • It is installed from the internal PyPI mirror and updated weekly.
  • nw doctor validates Git, Docker, registry auth, and vault access.
  • nw is required by most other project setup pages.

Setup Process

  1. Install nw from the internal mirror.
  2. Authenticate with SSO.
  3. Run nw doctor and resolve any red checks.

Commands

Run from a clean shell and keep the output with the setup or incident record.

pip install --index-url https://pypi.northwind.local/simple northwind-cli
nw login --sso
nw doctor

What To Keep

Keep the setup ticket, bootstrap output, verification log with the related ticket or record.
