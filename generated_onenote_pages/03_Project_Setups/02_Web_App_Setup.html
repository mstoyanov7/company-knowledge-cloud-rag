Web App Setup

Local setup for the customer web app (React/TypeScript/Vite).

Overview

Web App Setup is the project setup record maintained by Frontend Team for software engineers joining a project. It covers local setup for the customer web app (React/TypeScript/Vite). web-app is a React 19 + TypeScript app built with Vite.

Ownership

Owner teamFrontend Team
Audiencesoftware engineers joining a project
SystemsGitHub, Vite, API Gateway, Playwright

Key Facts

  • web-app is a React 19 + TypeScript app built with Vite.
  • It runs on port 5173 and proxies the API gateway at api.dev.northwind.local.
  • Component tests use Vitest; end-to-end tests use Playwright.
  • Design tokens come from the shared @northwind/ui package.

Setup Process

  1. Clone the repo and install dependencies with npm.
  2. Copy the example env file.
  3. Start the dev server and run the tests.

Commands

Run from a clean shell and keep the output with the setup or incident record.

git clone ssh://git.northwind.local/frontend/web-app.git
cd web-app && npm install
cp .env.example .env.local
npm run dev
npm run test && npm run e2e
