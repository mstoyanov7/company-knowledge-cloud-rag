Software and SaaS Purchasing

Buying new software tools and subscriptions.

Summary

Software and SaaS Purchasing is the finance process maintained by Procurement for managers, project owners, finance partners. It covers buying new software tools and subscriptions. Any new SaaS tool requires a security and data-privacy review before purchase.

Key Facts

  • Any new SaaS tool requires a security and data-privacy review before purchase.
  • Purchases over 5,000 EUR per year need finance director approval.
  • Prefer existing approved vendors before evaluating new ones.
  • Annual contracts are preferred over monthly for tools used by whole teams.

Approval Process

  2. Submit a purchase request with a business case.
  3. Complete the security review for new vendors.

What To Keep
